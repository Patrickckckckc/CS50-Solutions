# CS50 Web Programming with Python and JavaScript

This repository contains my solutions and projects from **CS50’s Web Programming with Python and JavaScript** by Harvard University (EdX).  
The course focuses on building dynamic web applications using **Flask**, **SQL**, **HTML**, **CSS**, and **JavaScript**, while introducing modern web development concepts such as APIs, authentication, and front‑end frameworks.

---

## 📚 Course Structure

Each folder represents a project from the course, progressively increasing in complexity:

| Project | Title | Description |
|----------|--------|-------------|
| 0 | Search | Introduction to HTML, CSS, and responsive design. |
| 1 | Wiki | A Wikipedia‑style site using Django, Markdown, and forms. |
| 2 | Commerce | E‑commerce platform with user authentication and bidding system. |
| 3 | Mail | Email client built with JavaScript and API integration. |
| 4 | Network | Social network application with posts, likes, and follows. |

---

## 🧩 Overview

Throughout these projects, I learned to:
- Build full‑stack web applications using **Python (Flask/Django)**.  
- Connect and query databases using **SQL**.  
- Implement **user authentication** and **session management**.  
- Use **JavaScript** for dynamic front‑end interactions.  
- Apply **HTML/CSS** for responsive and accessible design.
