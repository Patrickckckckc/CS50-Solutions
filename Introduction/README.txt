# CS50 Introduction to Computer Science

This repository contains my solutions and projects from **CS50x – Harvard University (via EdX)**.  
It covers the fundamentals of computer science, programming, and problem-solving using **C, Python, SQL, HTML, CSS, and JavaScript**.

---

## 📚 Course Overview

Each folder represents a week of the CS50 curriculum, focusing on different core concepts:

| Week | Topic | Description |
|------|--------|-------------|
| 1 | C Programming | Introduction to algorithms, loops, and conditionals. |
| 2 | Arrays | Manipulating data structures and memory. |
| 3 | Algorithms | Searching, sorting, and efficiency. |
| 4 | Memory | Pointers, dynamic memory, and debugging. |
| 5 | Data Structures | Linked lists, stacks, and queues. |
| 6 | Python | Transition from C to Python for higher-level programming. |
| 7 | SQL | Working with databases and queries. |
| 8 | HTML, CSS, JavaScript | Basics of web development and front-end design. |
| 9 | Flask | Building dynamic web applications using Python. |

---

## 🧘 Final Project: Web Programming Meditation App

For the final project, I developed a **Meditation Web Application** using **Flask** and **SQLite**.  
The app includes:
- Pages for different meditation modes (Normal and Letting Go).  
- User-friendly interface with CSS animations.  
- Dynamic content handled through JavaScript.  
- Database integration for storing user interactions.  

This project combines the skills learned throughout the course and demonstrates my ability to design, code, and deploy a complete web application.
