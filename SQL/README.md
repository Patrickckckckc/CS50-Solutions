# CS50 SQL Programming

This repository contains my solutions and projects from **CS50’s Introduction to Databases with SQL** by Harvard University (EdX).  
The course explores how to design, query, and manage relational databases using **SQL**, focusing on data modeling, normalization, and efficient querying.

---

## 📚 Course Structure

Each folder represents a week of the course, covering different aspects of SQL and database design:

| Week | Topic | Description |
|------|--------|-------------|
| 0 | Introduction to SQL | Basic syntax, tables, and queries. |
| 1 | Filtering and Sorting | Using `WHERE`, `ORDER BY`, and logical operators. |
| 2 | Aggregation | Applying `COUNT`, `SUM`, `AVG`, `MIN`, and `MAX`. |
| 3 | JOINs | Combining data from multiple tables using `INNER`, `LEFT`, and `RIGHT JOIN`. |
| 4 | Design | Understanding normalization, primary/foreign keys, and relationships. |
| 5 | Indexes and Views | Optimizing queries and creating reusable views. |
| 6 | Final Project | Applying all learned concepts in a complete database application. |

---

## 🧩 Final Project: SQL Data Application

My final project demonstrates the use of **SQL** to design and query a structured database.  
It includes:
- Creation of multiple related tables with **primary and foreign keys**.  
- Use of **JOINs** and **aggregations** to extract meaningful insights.  
- Implementation of **indexes** for performance optimization.  
- Queries that simulate real-world data analysis scenarios.
